# Dompet Digital Sample


## Description

Aplikasi sebagai untuk mengetahui kemampuan dari pemilik github

## Installation

### 1. Copy File .env.example ke .env dalam be
### 2. Jalankan perintah
```bash
go mod tidy
```

### 3. jalan perintah untuk menjalankan aplikasi
```bash
go run .
```
### 3.1 (optional) jika membutuhkan data dummy ada di "data_dummy.sql"
